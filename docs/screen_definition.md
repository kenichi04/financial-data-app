## SCR-001 トップ / 銘柄検索画面

### 概要

サービスのトップページ兼銘柄検索の起点。企業名または証券コードでの検索結果を一覧表示する。

### URL

`/`

### 遷移

| 操作 | 遷移先 |
|---|---|
| 検索結果の企業行をクリック | SCR-002 企業詳細画面 |

### 画面構成

```
[グローバルヘッダー]
────────────────────────────────────────
[ヒーローエリア]
  サービスキャッチコピー
  ─────────────────────
  [検索ボックス]  [検索ボタン]

[検索結果テーブル]（検索実行後に表示）
  企業名 | 証券コード | 業種 | ROE | ROA | 現金比率

[フッター]
```

### 入力項目

| 項目 | 種別 | バリデーション | 備考 |
|---|---|---|---|
| 検索キーワード | テキスト | 1文字以上 | 企業名（部分一致）または証券コード（前方一致）で検索 |

### 表示項目（検索結果一覧）

| 項目 | 説明 | 備考 |
|---|---|---|
| 企業名 | `company.name` | |
| 証券コード | `company.security_code` | |
| 直近 ROE | 最新期の ROE（%） | `financial_data` から都度計算 |
| 直近 ROA | 最新期の ROA（%） | |
| 直近 現金比率 | 最新期の現金比率（%） | |

### アクション

| アクション | 説明 |
|---|---|
| 検索ボタン押下 / Enter キー | 銘柄検索API を呼び出し、結果をテーブルに表示 |
| 検索結果行クリック | SCR-002 へ遷移 |

### API 連携

| API | メソッド | 備考 |
|---|---|---|
| 銘柄検索 API | GET `/api/companies?q={keyword}` | 企業名・証券コードで部分検索 |

### アクセス制御

- Phase 1：認証なし。全ユーザーアクセス可。
- Phase 2：未ログインでも利用可。

## SCR-002 企業詳細画面

### 概要

特定企業の財務指標一覧と時系列グラフを表示する。Phase 1 の中心的な画面。

### URL

`/company/[securityCode]`

例：`/company/7203`（トヨタ自動車）

### 遷移

| 操作 | 遷移先 |
|---|---|
| ヘッダーの検索 | SCR-001（検索結果表示） |
| 「ウォッチリストに追加」ボタン（Phase 2） | ウォッチリスト選択モーダル |

### 画面構成

```
[グローバルヘッダー]
────────────────────────────────────────
[企業ヘッダーエリア]
  企業名  証券コード  EDINETコード  決算期

[財務指標カードエリア]
  [ROE]  [ROA]  [営業利益率]  [自己資本比率]  [現金比率]
  ※Phase 2以降: [PER]  [PBR] を追加

[時系列グラフエリア]
  [指標セレクト] [期数セレクト（3期 / 5期 / 10期）]
  ─────────────────────────────
  折れ線グラフ（横軸：決算期、縦軸：指標値）

[ウォッチリスト追加ボタン]（Phase 2、ログイン＆プレミアム限定）

[フッター]
```

### 表示項目

#### 企業ヘッダー

| 項目 | 説明 |
|---|---|
| 企業名 | `company.name` |
| 証券コード | `company.security_code` |
| EDINETコード | `company.edinet_code` |
| 最新決算期末日 | `financial_document.fiscal_year_end_date`（最新） |

#### 財務指標カード（直近期）

| 指標 | 計算式 | Phase |
|---|---|---|
| ROE | 当期純利益 ÷ 自己資本 × 100（%） | 1 |
| ROA | 当期純利益 ÷ 総資産 × 100（%） | 1 |
| 営業利益率 | 営業利益 ÷ 売上高 × 100（%） | 1 |
| 自己資本比率 | 純資産 ÷ 総資産 × 100（%） | 1 |
| 現金比率 | 現金及び現金同等物 ÷ 総資産 × 100（%） | 1 |
| PER | 株価 ÷ EPS | 2 |
| PBR | 株価 ÷ BPS | 2 |

#### 時系列グラフ

- 横軸：決算期末日
- 縦軸：選択した指標の値（%）
- 表示対象期数：ドロップダウンで選択（3期 / 5期 / 10期）
- 連結・個別の区分：連結を優先。連結がない場合は個別を表示

### 入力項目

| 項目 | 種別 | 備考 |
|---|---|---|
| グラフ表示指標 | セレクトボックス | ROE / ROA / 営業利益率 / 自己資本比率 / 現金比率 |
| 表示期数 | セレクトボックス | 3期 / 5期 / 10期 |

### アクション

| アクション | 説明 |
|---|---|
| 指標 / 期数セレクト変更 | グラフを再描画（APIを再コール） |
| ウォッチリスト追加ボタン（Phase 2） | リスト選択モーダルを表示し、対象銘柄を登録 |

### API 連携

| API | メソッド | 備考 |
|---|---|---|
| 企業詳細取得 | GET `/api/companies/{securityCode}` | 企業基本情報 |
| 財務指標一覧取得 | GET `/api/companies/{securityCode}/metrics` | 直近期の指標カード用 |
| 財務指標時系列取得 | GET `/api/companies/{securityCode}/metrics/history?years={n}` | グラフ用 |

### アクセス制御

- Phase 1：認証なし。全指標を全ユーザーに公開。
- Phase 2：PER/PBR は全ユーザー閲覧可（株価API連携後）。ウォッチリスト追加はプレミアムのみ。

## SCR-003 ユーザー登録画面 ※Phase 2

### 概要

メールアドレスとパスワードで新規アカウントを作成する。

### URL

`/register`

### 遷移

| 操作 | 遷移先 |
|---|---|
| 登録成功 | SCR-005 マイページ |
| 「ログインはこちら」リンク | SCR-004 ログイン画面 |

### 画面構成

```
[グローバルヘッダー]
────────────────────────────────────────
[登録フォーム]
  メールアドレス
  パスワード
  パスワード（確認）
  表示名
  [登録ボタン]
  ─────────
  すでにアカウントをお持ちの方は[ログイン]

[フッター]
```

### 入力項目

| 項目 | 種別 | バリデーション |
|---|---|---|
| メールアドレス | メール入力 | 必須・メール形式 |
| パスワード | パスワード入力 | 必須・8文字以上 |
| パスワード（確認） | パスワード入力 | 必須・パスワードと一致 |
| 表示名 | テキスト入力 | 必須・50文字以内 |

### API 連携

| API | メソッド |
|---|---|
| ユーザー登録 | POST `/api/auth/register` |

## SCR-006 スクリーニング画面 ※Phase 2（プレミアム限定）

### 概要

複数の財務指標に条件を設定し、条件を満たす銘柄を絞り込む。

### URL

`/screening`

### アクセス制御

- 未ログイン：ログイン画面へリダイレクト
- ログイン済み（free）：画面表示はするが、プレミアム加入を促すバナーを表示し検索は不可
- プレミアム：全機能利用可

### 画面構成

```
[グローバルヘッダー]
────────────────────────────────────────
[条件設定パネル]
  ROE         [最小値] ～ [最大値]
  ROA         [最小値] ～ [最大値]
  営業利益率  [最小値] ～ [最大値]
  自己資本比率 [最小値] ～ [最大値]
  現金比率    [最小値] ～ [最大値]
  [検索ボタン]  [条件クリアボタン]

[検索結果テーブル]
  企業名 | 証券コード | ROE | ROA | 営業利益率 | 自己資本比率 | 現金比率
  ※各行クリックで SCR-002 へ遷移

[フッター]
```

### 入力項目

| 項目 | 種別 | バリデーション |
|---|---|---|
| ROE 最小値 / 最大値 | 数値入力 | 任意・数値 |
| ROA 最小値 / 最大値 | 数値入力 | 任意・数値 |
| 営業利益率 最小値 / 最大値 | 数値入力 | 任意・数値 |
| 自己資本比率 最小値 / 最大値 | 数値入力 | 任意・数値 |
| 現金比率 最小値 / 最大値 | 数値入力 | 任意・数値 |

### API 連携

| API | メソッド | 備考 |
|---|---|---|
| スクリーニング | GET `/api/screening?roeMin=&roeMax=&...` | `financial_metrics` テーブルを WHERE 検索 |

## SCR-007 ウォッチリスト画面 ※Phase 2（プレミアム限定）

### 概要

登録した銘柄のウォッチリストを管理・閲覧する。複数のリストを持てる。

### URL

`/watchlist`

### アクセス制御

- 未ログイン・free ユーザー：プレミアム加入促進画面を表示

### 画面構成

```
[グローバルヘッダー]
────────────────────────────────────────
[リストタブ]
  [リスト名A] [リスト名B] ... [+ 新規リスト作成]

[選択中リストの銘柄一覧]
  企業名 | 証券コード | ROE | ROA | 現金比率 | 追加日 | [削除]
  ※各行クリックで SCR-002 へ遷移

[フッター]
```

### アクション

| アクション | 説明 |
|---|---|
| 新規リスト作成 | リスト名入力モーダルを表示し `watchlists` にレコード追加 |
| リストタブ切替 | 選択リストの銘柄一覧を表示 |
| 銘柄行クリック | SCR-002 へ遷移 |
| 削除ボタン | 確認後、`watchlist_items` からレコード削除 |

### API 連携

| API | メソッド |
|---|---|
| ウォッチリスト一覧取得 | GET `/api/watchlists` |
| リスト内銘柄一覧取得 | GET `/api/watchlists/{id}/items` |
| 銘柄追加 | POST `/api/watchlists/{id}/items` |
| 銘柄削除 | DELETE `/api/watchlists/{id}/items/{companyId}` |
| リスト作成 | POST `/api/watchlists` |
| リスト削除 | DELETE `/api/watchlists/{id}` |

## SCR-008 サブスクリプション管理画面 ※Phase 2

### 概要

有料プランへの加入・解約・プラン状況の確認を行う。

### URL

`/mypage/subscription`

### 遷移

| 操作 | 遷移先 |
|---|---|
| プレミアム加入ボタン | Stripe Checkout（外部ページ） |
| 決済完了後のコールバック | SCR-005 マイページ |

### 表示項目

| 項目 | 説明 |
|---|---|
| 現在のプラン | free / premium |
| サブスク状態 | active / canceled / past_due |
| 現在の契約期間 | `subscriptions.current_period_start` ～ `current_period_end` |

### アクション

| アクション | 対象 | 説明 |
|---|---|---|
| プレミアムに加入 | free ユーザー | Stripe Checkout へリダイレクト |
| 解約 | premium ユーザー | 確認モーダル後 Stripe でキャンセル処理 |

### API 連携

| API | メソッド | 備考 |
|---|---|---|
| サブスク情報取得 | GET `/api/subscriptions/me` | |
| Checkout セッション作成 | POST `/api/subscriptions/checkout` | Stripe Checkout URL を返す |
| 解約 | POST `/api/subscriptions/cancel` | Stripe でキャンセル |

```
  └─ 検索結果クリック ─→ SCR-002（企業詳細）
                              └─ ウォッチリスト追加 ─→ [モーダル] ─→ SCR-007（ウォッチリスト）

  └─ 結果行クリック ─→ SCR-002（企業詳細）
```

## 備考

- Phase 1 は認証なし。SCR-001・SCR-002 のみ実装する。
- Phase 2 でユーザー認証・スクリーニング・ウォッチリスト・サブスクを追加する。
- PER・PBR は株価データ API 連携（Phase 2）後に SCR-002 の指標カードに追加する。
- 免責事項はフッターに常設し、投資判断はユーザー責任である旨を明記する。
